## المميزات ✨

- ✅ **مستقل تماماً** - لا يعتمد على تطبيق الهاتف
- ✅ **سهل الاستخدام** - واجهة بسيطة وواضحة
- ✅ **آمن** - لا يتم حفظ كلمات مرور
- ✅ **سريع** - معالجة فورية للطلبات
- ✅ **متعدد المستخدمين** - يدعم عدة مستخدمين في نفس الوقت
- 📊 **إحصائيات** - تتبع التسجيلات الناجحة

## الإعداد ⚙️

### 1. الحصول على Token البوت

1. افتح Telegram وابحث عن `@BotFather`
2. أرسل `/newbot`
3. اتبع التعليمات واحصل على Token
4. انسخ Token

### 2. تعيين Token

يمكنك تعبئة التوكين بأحد الطرق التالية:

- **متغير بيئة** (موصى به) – هذا هو السلوك الافتراضي بعد التحديث الأخير، ولا تحتاج لتعديل الكود:
  ```bash
  export TELEGRAM_BOT_TOKEN="your_token_here"
  # أو استخدم ملف `.env` (انظر أدناه)
  ```
- **تعديل الرمز** – إذا أردت حجز التوكين داخل الملف:
  ```python
  TOKEN = "YOUR_ACTUAL_BOT_TOKEN"
  ```

`djezzy_bot.py` الآن يقرأ قيمة المتغير `TELEGRAM_BOT_TOKEN` من البيئة ويوقف التنفيذ إذا لم يتم تعيينه أو كان تنسيقه غير صحيح. سكريبتات التشغيل (`run.sh`، `setup_token.sh`، `install.sh` وغيرها) ستتحقق من صحة الـ token كما تُبلغك إذا كان خالياً أو غير صالح.

### 3. (اختياري) استخدام ملف .env

يمكنك إنشاء نسخة من المثال وتعديلها:

```bash
cp .env.example .env
nano .env
# أضف سطر: TELEGRAM_BOT_TOKEN=your_actual_token
```

السكربتات `run.sh` و `setup_token.sh` يقومان بتحميل المحتويات تلقائياً عند التشغيل، لذلك لا تحتاج لتعديل الكود لتعريف المتغير.

## استكشاف الأخطاء 🔍

### الخطأ: "فشل إرسال الكود"

- ✓ تحقق من كتابة الرقم بشكل صحيح
- ✓ تأكد من توفر الإنترنت
- ✓ انتظر قليلاً وحاول مرة أخرى

### الخطأ: "فشلت جميع المحاولات"

- ✓ قد يكون الرقم مستخدماً كثيراً
- ✓ جرب لاحقاً
- ✓ استخدم رقم مختلف

### الخطأ: "Token غير صحيح"

- ✓ تحقق من Token من @BotFather
- ✓ تأكد من عدم وجود مسافات إضافية

## الأمان ⚠️

- ❌ لا تشارك Token البوت مع أحد
- ❌ لا تضع Token في ملفات عامة
- ✓ استخدم متغيرات البيئة للـ Token
- ✓ احذر عند مشاركة السجلات

## التطوير 👨‍💻

### هيكل الكود

**djezzy_utils.py:**

- وحدة مستقلة تماماً
- يمكن استخدامها في أي مشروع آخر
- توفر جميع الوظائف الأساسية

**djezzy_bot.py:**

- بوت Telegram كامل
- يستخدم `python-telegram-bot`
- يدير المحادثات والحالات

### إضافة ميزات جديدة

1. أضف الوظيفة في `djezzy_utils.py`
2. أضف المعالج في `djezzy_bot.py`
3. اختبر جيداً
